## 💡 專題報告

**主題:** 自動化病歷輔助生成系統  
**組員:** 郭秉龍 (組長)、李東壕、吳紹齊  
**指導教授:** 郭冠良醫師 (聯合醫院仁愛院區 家庭醫學科 專任主治醫師兼主任)  
**使用技術:** PHP、JS、XAMPP、HTML、CSS  

**專案背景:**  
醫療領域逐漸引入自動化技術，但病歷撰寫仍耗費醫師大量時間。此專案旨在透過 OpenAI 自然語言處理技術，提升效率並減少人為錯誤。

**主要成果:**  
- 提供快速生成病歷與建議治療方案的輔助工具。  
- 提升醫療效率與病歷精確性。
